# 13 — 交付本地个人词库

**What to build:** 用户可以维护本地标准词条、口语别名和启用状态；一次听写会快照启用词库、向豆包提供请求级上下文，并对唯一明确的别名做可追溯规范化。

**Blocked by:** 12 — 交付豆包默认顺滑听写

**Status:** ready-for-agent

- [ ] 设置页支持新增、编辑、启停和删除词条，所有数据只保存在本机
- [ ] 空标准写法、重复标准写法和多个启用词条争用同一别名时给出明确校验
- [ ] 按下快捷键时快照启用词条，录音期间的编辑不影响当前会话
- [ ] 请求级 context 遵守供应商容量限制并使用确定性截断策略
- [ ] 本地只规范化唯一、完整匹配的别名，不误改普通子串
- [ ] 会话记录能说明词库快照与实际规范化结果，公开 seam 测试通过
